# analitica-precio-del-cafe

Este proyecto tiene como objetivo analizar el comportamiento y la volatilidad del precio internacional del café mediante el uso de analítica de datos aplicada a series de tiempo mensuales. Se busca identificar tendencias, ciclos y patrones, así como evaluar la relación entre el precio del café y variables económicas como el índice del dólar, el precio del petróleo y los costos de producción.
